# 🌱 AgroEngine — Deploy no GitHub Pages

## Passo a passo COMPLETO (10 minutos, gratuito)

---

## 1️⃣ Criar conta no GitHub (se não tiver)

1. Acesse **github.com** no seu computador
2. Clique em **"Sign up"**
3. Preencha: email, senha, username (ex: `carlos-agro`)
4. Confirme o email

---

## 2️⃣ Criar o repositório

1. Logado no GitHub, clique no **"+"** (canto superior direito)
2. Clique em **"New repository"**
3. Preencha:
   - **Repository name:** `agroengine`
   - **Description:** Meu agrônomo digital
   - Marque **"Public"** ✅
   - Marque **"Add a README file"** ✅
4. Clique em **"Create repository"**

---

## 3️⃣ Fazer upload dos arquivos

1. Na página do repositório, clique em **"Add file"** → **"Upload files"**
2. Arraste os 4 arquivos desta pasta:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `README.md`
3. Clique em **"Commit changes"**

---

## 4️⃣ Ativar o GitHub Pages

1. No repositório, clique em **"Settings"** (engrenagem)
2. No menu esquerdo, clique em **"Pages"**
3. Em **"Source"**, selecione:
   - Branch: **main**
   - Folder: **/ (root)**
4. Clique em **"Save"**
5. Aguarde ~2 minutos

---

## 5️⃣ Acessar o app online

Seu link será:
```
https://SEU-USUARIO.github.io/agroengine
```

Exemplo: `https://carlos-agro.github.io/agroengine`

---

## 📱 Instalar no celular (opcional mas recomendado)

### Android (Chrome):
1. Abra o link no Chrome
2. Toque no menu **⋮**
3. Toque em **"Adicionar à tela inicial"**
4. Confirme — vira um ícone igual a um app! 🎉

### iPhone (Safari):
1. Abra o link no Safari
2. Toque no botão **Compartilhar** (quadrado com seta)
3. Toque em **"Adicionar à Tela de Início"**
4. Confirme

## ✏️ Atualizar o app no futuro

1. Abra o repositório no GitHub
2. Clique no arquivo que quer atualizar
3. Clique no lápis ✏️ para editar
4. Ou clique em **"Add file"** → **"Upload files"** para substituir

---

## ❓ Problemas?

- **Página não carrega:** Aguarde mais 5 minutos após ativar Pages
- **App sem funcionalidades:** Verifique se o arquivo é `index.html` (não `index.html.txt`)
- **Link não funciona:** Confirme que o repositório é **Public**
